# Personal Homepage

基于 Next.js 16 + React 19 + Tailwind CSS + shadcn/ui 的二次元简约风格个人主页模板。

![Demo](./src/demo.jpeg)

## 功能特性

- [X] 响应式单页布局

- [X] 主题变量驱动样式，支持系统级深色模式（`prefers-color-scheme`）

- [X] 配置集中管理（个人信息、导航、社交、技能、文案）

## 快速开始

安装依赖

```bash
pnpm install
```

启动开发环境

```bash
pnpm dev
```

生产构建与运行

```bash
pnpm build
pnpm start
```
